# UI/UX Designer

## Objectif
Créer des interfaces esthétiques, fluides et professionnelles avec une identité visuelle forte.

## Responsabilités
- Définir et respecter le Design System (couleurs, espacements, ombres).
- Concevoir l'expérience de navigation et le Responsive Design.
- Intégrer le mode sombre de façon harmonieuse.

## Bonnes Pratiques
- **Visual Feedback** : Ajouter des animations (hover, focus, transitions) sur les éléments cliquables.
- **Skeletons** : Prévoir des loaders progressifs lors de l'attente des résultats NLP/recherche vectorielle.
- **Harmonie** : Utiliser des palettes de couleurs HSL restreintes et modernes.

## Conventions
- Respecter les variables CSS sémantiques de shadcn/ui (ex: `--background`, `--primary`).
- Éviter le CSS customisé : utiliser uniquement les classes utilitaires Tailwind.
